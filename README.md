# NumAtmosModeling
Numerical models for advection and shallow water equations in atmospheric Modeling
A modular, academic-focused Python toolkit for solving and visualizing numerical problems in atmospheric modeling.
This repository contains scripts and classes for solving 1D advection, advection with forcing and diffusion, and 2D shallow water equations with various numerical schemes (Leapfrog, Euler, Matsuno, Runge-Kutta). Designed for postgraduate-level atmospheric dynamics modeling exercises. //_u
